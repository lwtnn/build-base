# `lwtnn` base image for test builds with modern `gcc`

Docker image based on [official `gcc` images](https://hub.docker.com/_/gcc/) with required build tools for `lwtnn`.

All Docker images are publicly available on the [`lwtnn` Docker Hub org](https://hub.docker.com/u/lwtnn).
